# Научный клубок — руководство пользователя

## Роли

| Роль | Что видит/может |
|---|---|
| **researcher** | запросы, граф, evidence, экспорт (без restricted источников по чужой политике) |
| **analyst** | то же + сравнительный анализ, дашборды |
| **project_manager** | + дашборды покрытия, аудит, scheduled-отчёты |
| **curator** | + правка графа, review-очередь, merge/split, история решений |
| **external_partner** | ограниченный доступ: без внутренних/коммерческих источников и контактов экспертов |
| **admin** | всё + управление, аудит-лог |

Вход: `POST /api/v1/auth/login {username, role}` → JWT; передавайте `Authorization: Bearer <jwt>`.

## Экран «Запрос» (researcher)

Сформулируйте инженерный вопрос на русском или английском — можно смешивать
материал + процесс + условия + числа + географию + период. Пример:

> «Какие методы обессоливания воды подходят, если сульфаты/хлориды/Ca/Mg/Na по
> 200–300 мг/л, а сухой остаток ≤1000 мг/дм³?»

Ответ содержит:
- **Достоверность** (шкала 0–100%) — среднее по доказательной базе с поправкой на
  противоречия.
- **Текст** со ссылками `[n]` — каждое утверждение привязано к источнику.
- **Таблица сравнения** решений (практика, ключевые метрики, применимость).
- **⚠️ Противоречия** и **🔍 Пробелы** — явно выделены, не «усредняются».
- **Источники** — клик открывает **Evidence Inspector** (документ, страница, цитата,
  сила доказательства, статус верификации).
- **Клубок** справа — подграф ответа: узлы (материал/процесс/оборудование/…) и
  связи с доказательствами.

### Визуальные сигналы графа
- Медь — технологии/решения; сталь-никель — материалы; янтарный (пунктир, полый) —
  пробел; красный (пунктир) — противоречие; зелёное кольцо — verified.

## Экран «Сравнение» (analyst)

Введите набор технологий/условие — получите нормализованную таблицу: каждая ячейка
либо число+единица со значком доказательства, либо явный «пробел» (никогда пусто).

## Экран «Покрытие» (manager)

Метрики знаний по направлениям (гидро/пиро-металлургия, экология, водоподготовка…):
источники, решения, измерения, пробелы, противоречия, зоны риска. Клик → детали.

## Экран «Пробелы и риски» (curator/analyst)

Пробелы, противоречия и очередь ревью рядом — рабочее место верификации знаний.

## Экран «Глоссарий»

Единый RU/EN словарь: `ПВП` = `печь взвешенной плавки` = `flash smelting furnace`.

## Курирование (curator)

- Правка сущности/связи/числа/единицы/географии/применимости → создаётся
  `CurationEvent` (кто, когда, было→стало, причина). Исправленные поля **не
  перезатираются** при повторной загрузке.
- `merge` дубликатов, `add alias`, accept/reject, история изменений на карточке.

## Экспорт и уведомления

- Экспорт ответа: Markdown / JSON-LD (`/api/v1/export`).
- Подписка на тему: `/api/v1/notifications/subscribe {topic}` → оповещения о новых
  источниках, противоречиях, пробелах (с учётом прав доступа).

## FAQ

- **Нет ответа/источников?** Возможно, в корпусе нет данных — смотрите блок
  «Пробелы»; попробуйте переформулировать или снять числовые ограничения.
- **Что такое «пробел»?** Комбинация материал×процесс×условие без экспериментов.
- **Числам можно доверять?** Каждое число нормализовано (pint) и привязано к
  цитате — проверяйте через Evidence Inspector.
